ABOUT
-----
This directory contains _xkbswitch_,
a tiny keyboard layout switcher for X.

REQUIREMENTS
------------
Build time:
  * c99 compiler
  * POSIX sh(1p), make(1p) and "mandatory utilities"
  * libX11
